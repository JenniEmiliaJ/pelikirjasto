Mobiiliohjelmointi kurssin lopputyö. Aiheena pelikirjasto lautapelikerholle, jolla voidaan selata pelikerholla pelattavissa olevia pelejä, lisätä, poistaa ja muokata pelejä sekä etsiä sopivaa peliä hakuehdoilla.

Kuvaus käytetyistä teknologioista:

- Kehitysympäristönä toiminut Expo ja mobiililaitteessa käyttöä testattu Expo Go sovelluksella.
- Firebase realtime database pelin tietojen tallentamiseen
- Firebase storage kuvien tallentamiseen
- Navigoinnissa stack-navigointi ja bottom-tab navigointi
- Ulkoasussa react-native-paper komponentit
